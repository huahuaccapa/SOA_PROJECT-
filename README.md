# 🛒 TechStore Frontend

Frontend de una tienda virtual desarrollado con React + Vite.  
Incluye dashboard administrativo, catálogo de productos, carrito de compras, sistema de notificaciones y gestión de pagos.

---

## 🚀 Tecnologías utilizadas

- ⚛️ React
- ⚡ Vite
- 🧭 React Router DOM
- 🎨 CSS personalizado
- 🧩 Lucide React (iconos)

---

## 📦 Funcionalidades principales

### 👤 Roles de usuario
- **ADMIN**
  - Dashboard con métricas
  - Gestión de productos, pedidos y usuarios
  - Reportes y finanzas

- **USER**
  - Visualización de pedidos
  - Carrito de compras
  - Lista de deseos
  - Gestión de perfil y direcciones
  - Métodos de pago

- **GUEST**
  - Navegación por catálogo
  - Acceso a carrito
  - Registro e inicio de sesión

---

### 🛍️ Catálogo
- Búsqueda de productos
- Filtros por categoría y precio
- Ordenamiento (precio, rating, nombre)
- Vista grid y lista
- Sistema de wishlist

---

### 🛒 Carrito de compras
- Agregar / eliminar productos
- Modificar cantidades
- Aplicar cupones de descuento
- Cálculo automático de:
  - Subtotal
  - Descuentos
  - Envío
  - Total
- Validación antes de pago

---

### 💳 Pagos
- Gestión de tarjetas
- Historial de transacciones
- Vista diferenciada para ADMIN y USER

---

### 🔔 Notificaciones
- Notificaciones dinámicas:
  - Pedidos
  - Pagos
  - Stock
- Marcar como leídas
- Filtrado por tipo

---

### 📦 Pedidos
- Visualización de pedidos
- Estados:
  - Pendiente
  - En proceso
  - Enviado
  - Entregado
- Seguimiento (tracking)

---

### 📊 Dashboard (ADMIN)
- Estadísticas:
  - Ventas
  - Pedidos
  - Usuarios
- Tablas dinámicas
- Indicadores visuales
